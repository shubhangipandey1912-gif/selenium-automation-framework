# Selenium Automation Framework

This project is a scalable UI automation framework built using:

Tech Stack:
- Java
- Selenium WebDriver
- Cucumber BDD
- Maven
- Page Object Model
- Extent Reports

Features:
- Modular framework design
- Driver manager
- Screenshot capture
- CI integration using GitHub Actions
- BDD feature files

Project Structure:

src/main/java
   ├── core
   ├── pages
   ├── utils
   └── reporting

src/test/java
   ├── runners
   ├── hooks
   └── stepdefinitions
